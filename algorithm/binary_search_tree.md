## 二叉查找树
### 特征
1. 左子树上所有的节点的值均小于或等于他的根节点的值
2. 右子树上所有的节点的值均大于或等于他的根节点的值
3. 左右子树也一定分别为二叉排序树

## 平衡二叉查找树-红黑树
### 特征
1. 根节点是黑色
2. 节点是红色或者黑色
3. 每个叶子节点都是黑色的空节点(NULL)
4. 每个红色节点的两个子节点都是黑色
5. 从任意节点到其每个叶子的所有路径都包含相同的黑色节点

### 左旋过程
1. 右子变新爹
2. 新爹的左子变老爹的右子
3. 老爹(旋转节点)变新爹的左子

### 右旋过程
1. 左子变新爹
2. 新爹的右子变老爹的左子
3. 老爹变新爹的右子
