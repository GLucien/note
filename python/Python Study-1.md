1. 直接运行py文件
	
	方法是在py文件的第一行加上一个特殊的注释：

	```
	#!/usr/bin/env python3

	print('hello, world')
	```
	
	然后，通过命令`chmod a+x xxx.py`给py文件以执行权限
	
	最后通过./xxx.py即可执行
	
	注： 只可以在Linux和Mac系统下执行
